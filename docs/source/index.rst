Welcome to InmetPy's documentation!
===================================

**InmetPy** is a Python Client Library to consume the (Brazilian) National Institute of Meteorology - **INMET** API, 
with data from more than 700 meteorological stations spread across Brazil.

.. note::

   This project is under active development.
